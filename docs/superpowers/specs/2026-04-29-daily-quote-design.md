# 每日名言/古诗词（可折叠展示）设计

## 背景与目标

首页标题下方当前展示固定文案。希望替换为“每日名言/古诗词”，每日自动变化，并以“内容---作者或出处”的格式展示，同时提供“思想哲理概括”的可折叠展开。

目标：

- 前端不再展示固定文案，改为“每日一句”组件
- 每日刷新（以北京时间 Asia/Shanghai 为准）
- 展示格式：
  - 默认：`内容---作者或出处`
  - 展开：`思想哲理概括`
- 每日内容由后端统一提供：`GET /api/daily_quote`

非目标：

- 接入第三方每日一句 API
- 复杂后台管理（增删改名言库）

## API 设计

### `GET /api/daily_quote`

返回：

```json
{
  "date": "2026-04-29",
  "text": "学而不思则罔，思而不学则殆。",
  "source": "《论语·为政》",
  "summary": "强调学习与思考要结合，才能真正掌握知识。"
}
```

规则：

- 以北京时间当日日期 `YYYY-MM-DD` 作为轮换种子
- 后端内置名言/古诗词列表，按 `hash(date) % N` 稳定选取

## 前端交互

- 页面加载时调用 `/api/daily_quote`
- 使用 `<details><summary>...</summary>...</details>` 实现可折叠
- 为减少重复请求，可使用 localStorage 缓存当天结果（缓存 key 包含 date）
- 若接口请求失败，展示一条内置兜底文案
