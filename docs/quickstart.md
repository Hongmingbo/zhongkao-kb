# 快速开始（3 分钟）

中考知识库是一个用于“资料沉淀 + 自动整理 + 高频检索”的轻量服务：支持上传资料（含 PDF/Word/图片/ZIP）、自动归类、打标筛选、全文搜索、拆题切块。

## 适合谁

- 学生：把零散资料沉淀成可检索的个人题库/知识库
- 家长/老师：按年级/地区/题型管理资料，快速查找与复盘

## 你会得到什么

- 一个网页：登录后上传资料，自动按学科整理
- 一套 API：检索、筛选、拆题、管理文件
- 多用户隔离：每个账号一套私有知识库

## 开始使用

1. 打开前端页面
   - Cloudflare Pages：`https://zhongkao-kb.pages.dev/`
2. 注册/登录
   - 注册需要邀请码（由部署方设置 `INVITE_CODE`）
3. 上传资料
   - 支持：`txt/md/pdf/docx/jpg/png/webp/zip`
4. 高频使用
   - 搜索：关键词全文搜索
   - 筛选：按年份/地区/类型/后缀快速筛选
   - 拆题：把试卷按题目切块，便于复习与刷题
